Emdidi

Match pictures

Objective:

The goal is to match two identical images before the time. Every matched picture is ten (10) points. The players can see their score increased by ten points every time two pictures match. When the players find all the identical images, they move to the next level. Each level determines the difficulty of the game. When they lose, the game is over. They can Restart the game in the beginning or exit(create or load a new username).

Instruction

Start the game

  1. Enter a username
  2. Click on the Save button: A message will display if the username is invalid.
  3. Click on the Start button: to run the game.

Register new username

  1. Click on New
  2. Enter username
  3. Click on the Save button: a message will display if the username is invalid

Load username

  1. Enter username
  2. Click on the Load button: A message will display if the username is invalid
  3. All the score info will automatically load

Pause / Resume game

  1. The Pause button will be disabled when the game has not started
  2. When the game begins, the Pause button is enabled
  3. Click on the Pause button: the timer pauses, and the cards do not flip
  4. Click on Resume button: to continue the game

Stop Game

  1. The Stop button will be disabled when the game has not started yet
  2. When the game begins, the Stop button is enabled
  3. Click on the Stop button: to stop the game and remove all the points earned during the game.
